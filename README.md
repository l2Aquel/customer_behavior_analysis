# Customer Shopping Behavior Analysis
This project analyzes 3,900 customer purchase records to uncover insights into spending behavior, product performance, customer segmentation, and subscription trends. The objective is to generate data-driven business recommendations using Python, PostgreSQL (SQL), and Power BI.

## 1. Project Title / Headline
An end-to-end retail analytics project analyzing customer shopping behavior using Python, PostgreSQL, and Power BI. The project uncovers revenue trends, customer segmentation patterns, subscription impact, and product performance insights from 3,900 purchase records.

## 2. Short Description / Purpose
This project simulates real-world retail business analysis by combining data preprocessing, SQL-based business querying, and interactive dashboard visualization.
The objective is to identify revenue drivers, loyal customers, discount impact, and operational opportunities to support data-driven decision-making.

## 3. Tech Stack
- Python (Pandas, NumPy) – Data cleaning & feature engineering
- PostgreSQL – Business analysis & SQL views
- Power BI Desktop – Dashboard visualization
- P ower Query & DAX – Data modeling and KPI calculations
- File Types – .ipynb / .py, .sql, .pbix, .pdf

## 4. Data Source
- 3,900 transactional records
- 18 columns
- 37 missing values (Review Rating imputed using category median)

### Key Fields
- Customer: Age, Gender, Location, Subscription Status
- Purchase: Item, Category, Amount, Season
- Behavior: Discount Applied, Previous Purchases, Frequency, Review Rating, Shipping Type

## 5. Project Highlights
### Data Preparation (Python)
- Cleaned and standardized dataset
- Imputed missing values
- Created derived features (age_group, purchase frequency)
- Loaded processed data into PostgreSQL

### SQL Business Analysis
- Revenue by gender and age group
- Subscriber vs non-subscriber comparison
- High-spending discount users
- Customer segmentation (New, Returning, Loyal)
- Top-rated and best-selling products
- Shipping type comparison

### Power BI Dashboard
- Revenue distribution visuals
- Customer segmentation charts
- Product performance metrics
- Subscription and shipping insights
- KPI cards for business tracking

## 6. Key Insights
- Male customers generated ~68% of total revenue.
- Clothing is the highest revenue-generating category (~$100K).
- 27% of customers are subscribers.
- Express shipping users show higher average purchase value.
- Loyal customers contribute significantly to overall revenue.

## 7. Business Impact
- Opportunity to expand subscription programs.
- Strengthen loyalty strategies for repeat buyers.
- Optimize discount policy to balance sales and margins.
- Target high-revenue segments with focused marketing campaigns.

## 8. Conclusion
This project demonstrates structured data cleaning, SQL-driven business analysis, dashboard development, and practical retail analytics aligned with real-world business intelligence requirements.
